## UV Dashboard 数据标签防重叠优化

- 三个业务趋势图表的数据标签在各类视窗下仍有重叠问题
- 重写了 dataLabelPlugin（index.html + share_page.html），采用四阶段防重叠策略：
  1. Phase A：段内标签（堆叠图内嵌标签，固定位置，不参与碰撞检测）
  2. Phase B：收集所有外部标签（柱顶 + 折线点）到统一数组
  3. Phase C：全局碰撞解决 — 线标签先试上方→翻转至下方→垂直偏移→跳过；柱标签优先放置
  4. Phase D：绘制所有已解决位置的标签
- 新增密度控制：`lineLabelIndices()` 根据每点像素宽度(pxPerPt)决定折线标签频率
  - pxPerPt≥75：每点都标；≥50：隔2点；≥35：隔3点；<35：隔4点
  - 始终标注首末点、最大最小值点
- 图表高度从180px提升至200px，给标签更多空间
- 关键文件：templates/index.html, templates/share_page.html
