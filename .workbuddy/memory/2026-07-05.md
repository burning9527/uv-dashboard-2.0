## UV Dashboard 分享卡片 — 顾问维度（独立计算）
- **业务定义**: 应续费 = 当期在读 - 当期开课前续报；顾问续费 = 当期开课后新增续费
- **当期判断**: 按今天日期对照期次开课日（零期6/29、一期7/13、二期7/27、三期8/10）
- **基线 run**: 取当期开课前最近一次校准 run 作为基线；若无（如零期是最早期次），取最早可用 run 近似
- **续报时点判定**: 对比基线 run 与最新 run 的 `continue_fall` 字段——基线已续=开课前续报；最新已续但基线未续=开课后新增
- **聚合维度**: UV（按学员去重），因为顾问对学员负责而非对科目
- **后端实现** (`app.py` `/api/share-card`): 新增 `by_advisor` 列表 + `advisor_current_period` 字段
- **前端实现**: 分享卡片新增「顾问续报（当期：零期）」表格，列：顾问/在读/开课前续报/应续费/顾问续费/续报率/进度条，附公式说明 badge
- **数据验证**: 13位顾问，零期数据合理（如戎凯01: 23在读/5开课前/18应续费/3顾问续费/16.7%）
