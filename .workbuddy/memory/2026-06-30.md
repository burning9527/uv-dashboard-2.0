  - 将期次 chip 改为 `div` 结构，使用 `onclick="onPeriodClick('一期')"` 显式处理点击事件
  - 新增 `onPeriodClick(period)`：点击「全部」清空选中；点击具体期次则切换其选中状态
  - 更新 `updatePeriodChips()` 仅根据 `currentPeriods` 设置 `.active` 类，不再依赖 checkbox 状态
  - 移除不再需要的 `.period-chip span { pointer-events: none; }` 样式
- **文件改动**：`uv-dashboard/templates/index.html`
- **效果**：期次筛选现在可正常单选/多选，班级卡片会联动刷新。
