## 2.0系统排班看板点击交互修复 + 卡片UI重设计

### 点击交互修复（事件委托替代内联onclick）
- **根本原因**: 原代码使用 `onclick="UVModal.openClassDetail('${c.class_id}')"` 和 `onclick="UVModal.openStudentDetail('${s.uid}')"` 内联事件绑定，在动态HTML中失效
- **修复方式**: 改用事件委托模式，在 `container.onclick` 中统一处理3种点击事件
  - 班级卡片点击 → `e.target.closest('.schedule-class-card')` → `card.dataset.classId` → `UVModal.openClassDetail(classId)`
  - 学员标签点击 → `e.target.closest('.student-tag.clickable')` → `tag.dataset.uid` → `UVModal.openStudentDetail(uid)`
