
## UV Dashboard 2.0 Windows 打包配置

### 背景
用户要求把 2.0 系统封装成 Windows .exe 安装包。PyInstaller 不支持跨平台编译，必须在 Windows 上打包。

### 创建的文件
1. `build_assets/build_assets/AppIcon.ico` — Windows .ico 图标（6 尺寸，111KB）
2. `uv_dashboard2_win.spec` — Windows PyInstaller 配置（.ico 图标、winforms/edgechromium backend、排除 macOS-only 模块）
3. `requirements.txt` — Python 依赖清单
4. `build_windows.bat` — 一键打包脚本（自动装依赖 + PyInstaller）
5. `installer.iss` — Inno Setup 安装包脚本（安装向导 + 桌面快捷方式 + 卸载）
6. `BUILD_WINDOWS.md` — 完整打包指南

### Windows 兼容性
- `app.py` 已兼容：`sys.platform == 'win32'` → `%APPDATA%\UV Dashboard 2.0\` 数据目录
- pywebview Windows backend：`webview.platforms.winforms` + `webview.platforms.edgechromium`
- WebView2 Runtime：Win11 自带，Win10 需额外安装

### 打包流程（Windows 上）
1. 拷贝 `uv-dashboard-2.0/` 到 Windows
2. 双击 `build_windows.bat`（或手动 pip install + pyinstaller）
3. 产物：`dist\UV Dashboard 2.0\UV Dashboard 2.0.exe`
4. 用 Inno Setup 编译 `installer.iss` → 正式安装包
